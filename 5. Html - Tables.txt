Table Structure:

Caption - Heading for the table
Header - thead, header for the table
Body - tr table row, td table cell
Footer - tfooter, footer for the table

Colspan and Rowspan:

When you want to merge column use colspan
when you want to merge rows use rowspan

Table Formatting:

Border - Creates border for table, rows and columns 

width/align - width to specify width in pixel for column and is specified in th section
              align is used to align left or right

Cellspacing - to create spacing between two cells

Cellpadding - to create left, right, top, bottom spacing within cell

nowrap - no wrap of cell text.
